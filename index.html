Uvod u HTML 

HTML  je jezik za obilježavanje koji koristimo za izradu web stranica.
Njime definiramo strukturu i sadržaj web stranica.

Povijest HTML-a

HTML nastao je početkom 1990-ih godina.

Uređivanje teksta

Tekst je najčešći sadržaj web stranica.
Da bi dobio odgovarajuću ulogu na web stranici i bio prikazan na određeni 
način uređujemo ga  HTML-om.

HTML atributi

Unutar početne oznake elemenata, nakon naziva elementa, po potrebi pišemo atribute.
Atributi interetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim je 
informacijama riječ.
Atribut se sastoji od imena i vrijednosti atributa.
